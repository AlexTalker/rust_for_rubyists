What Is This?

Orkestrix is a system I'm developing to make it easier to write about music and publish it.

How Does It Work?

Orkestrix uses Dexy, reStructuredText, and ABC Music Notation to let me write documents about music and code that output to HTML and PDF. This page is an example of using Orkestrix.
